# MACLOTHES

Link de la página: https://maclothes-m.netlify.app/

## ✨ Características

### 🏠 Home Page
- Banner interactivo con carrusel y pestañas Hombre/Mujer
- Slider infinito de marcas
- Sección de productos destacados (grid con "Ver más" / "Ver menos")
- Sección de ofertas con scroll horizontal en mobile
- Testimonios de clientes en slider horizontal

### 👕 Tienda y Productos
- Categorías para Hombre y Mujer con filtros dinámicos
- Filtros por subcategoría, talle, color y rango de precios
- Panel de filtros desplegable en mobile
- Página de detalle con galería de imágenes, selectores de talle/color
- Productos relacionados en slider horizontal

### 🛒 Carrito de Compras
- Persistencia con localStorage
- Modificar cantidades y eliminar productos
- Resumen de compra en tiempo real
- Contador visible en el header

### ❤️ Wishlist / Favoritos
- Guardar productos favoritos
- Persistencia con localStorage
- Página dedicada para ver y gestionar favoritos
- Botón de corazón animado en todas las cards

### 📦 Checkout
- Formulario de 4 pasos (datos personales, envío, pago, confirmación)
- Opciones de envío estándar/express
- Métodos de pago (efectivo / transferencia)
- Resumen de pedido en tiempo real
- Página de agradecimiento con número de pedido

### 👤 Autenticación
- Registro y login de usuarios
- Perfil de usuario editable
- Rutas protegidas (checkout, perfil, pedidos)
- Persistencia con localStorage (simulación de backend)

### 📋 Mis Pedidos
- Historial completo de compras
- Detalle de cada pedido con productos, precios e información de envío
- Estados de pedido (pendiente, en proceso, enviado, entregado)

### 🔍 Buscador
- Autocompletado con imágenes y nombres de productos
- Página de resultados de búsqueda
- Búsqueda en nombre, categoría y descripción

### 📱 Diseño Responsive
- Enfoque mobile-first
- Adaptación perfecta a móvil, tablet y desktop
- Sliders horizontales en mobile
- Grids responsivos en tablet y desktop

## 🛠️ Tecnologías utilizadas

| **React** - Biblioteca principal para la interfaz de usuario |
| **Vite** - Herramienta de desarrollo rápida |
| **React Router DOM** - Navegación y rutas dinámicas |
| **Sass** - Estilos con variables, mixins y módulos |
| **React Icons** - Biblioteca de íconos |
| **localStorage** - Persistencia de datos (carrito, wishlist, usuarios) |
